Prophet 6 Sound Librarian

This is a Java Swing application developed for Sequential Circuits Prophet 6 owners in mind. It allows user to connect Prophet 6, read and write data to the Prophet 6, and organize, rename and merge patch data.

The application jar file can be downloaded for free from the Github releases page here.
